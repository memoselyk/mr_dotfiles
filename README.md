# Dotfiles managed by VCSH

## Requirements

* `git`
* `vcsh`
* `mr`

## Testing on docker

The following commands can be used to test a fresh setup on an ubuntu system

```
docker build -t vcsh-home-test .
docker run -ti vcsh-home-test
```
